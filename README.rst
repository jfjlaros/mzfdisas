MZF Disassembler
================

This is a utility for retrieving information from an MZF image.

For editing or assembling, see AS2MZF_.

Please see ReadTheDocs_ for the latest documentation.


.. _AS2MZF: https://web.archive.org/web/20170831235615/http://www.sharpmz.org/mz-700/as2mzf.htm
.. _ReadTheDocs: https://mzfdisas.readthedocs.io/en/latest/index.html
